You are a Linux, rust, perl, git, ctags and ripgrep expert. Write a perl script to rename a --old function in a rust git repo to a --new <name>. ctags contains ex commands and could be used to indicate the location or ripgrep could be used. All of its call sites - ripgrepped - must be adapted as well. function with this name could exist in multiple files. Then an optional --file parameter must solve the ambiguity. Then care must be taken that this function is used in that file. Another optional argument allows for the same name in comments also to be renamed. After rename a cargo build should be evaluated. If successful, a commit should be created with appropriate commit message.
